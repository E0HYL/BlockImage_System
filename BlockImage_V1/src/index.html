相关技术



区块链

区块链技术是首个自带对账功能的数字记账技术实现，本质上是一种通过去中心化、共识信任的方式集体维护的特殊分布式数据库。整个网络不存在中心机构，各节点间权利义务对等，任意节点的损坏或灭失都不会威胁整个系统的正常运作；遵循公开透明的运行规则，所有数据的记录与传递使用非对称加密和哈希算法来确保真实性，从而促成系统各节点间达成一致信任；数据块由系统所有节点共同维护，运用分布式数据库技术使每个节点都能及时获得一份完整数据拷贝。

智能合约

Smart Contract是运行在可复制、共享的账本上的计算机程序，可以处理信息，接收、储存和发送价值。这个程序就像一个可以被信任的人，可以临时保管资产，总是按照事先的规则执行操作。智能合约一旦设立指定后，能够无需中介的参与自动执行，并且没有人可以阻止它的运行。区块链为智能合约提供可信执行环境，而智能合约为区块链扩展应用




数字水印

数字水印技术是指在数字作品中嵌入有关作者的版权信息，但不影响原始载体的使用价值。数字水印分为可见水印与不可见水印两种，其中不可见水印指水印以数字数据的方式加入数字作品中，但是无法被人体感知系统察觉。不可见水印主要用于版权保护领域，通过嵌入数字作品中的水印信息来避免或者阻止未经授权的复制和拷贝

感知哈希

感知哈希是多媒体数据集到感知摘要集的一类单向映射，它利用多媒体数字信息包含的特征来生成一组指纹信息，根据生成的数字指纹，可以用来做多媒体内容的识别、检索和认证工作。由于感知哈希算法具有感知鲁棒性和安全性，所以对比于传统的单向哈希函数更适合用于多媒体信息管理和保护领域




密码算法

SM2:国标推荐的公钥密码算法，实现安全的密钥交换。

SM3:国标推荐的杂凑算法，敏感信息上链时以哈希值存储，实现保密性的同时具有不可抵赖性。

SM4:国标推荐的公钥密码算法，加密存储在IPFS上的原图，保障图像所有者权益，在发生版权交易时保障买家权益。

IPFS

IPFS（InterPlanetary File System）是一个面向全球的、点对点的分布式文件系统，目标是为了补充（甚至取代）目前统治互联网的超文本传输协议（HTTP），将所有具有相同文件系统的计算设备连接在一起。它是一个开放源代码项目，原理上用基于内容的地址替代基于域名的地址，用户寻找的不是某个地址而是储存在某个地方的内容，不需要验证发送者的身份，而只需要验证内容的哈希，让网页速度更快、更安全、更健壮、更持久
